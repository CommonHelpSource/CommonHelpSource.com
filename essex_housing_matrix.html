Essex County Housing Matrix

Low-Income Housing

  1. Apply to CHAMP: https://www.mass.gov/apply-for-public-housing
  2. Create an email address if needed to apply
  3. Submit ID and income documentation
  4. Follow up with local Housing Authorities (Lawrence, Lynn, Salem)
  5. Check CHAMP status online regularly

Market-Rate Rentals

  1. Use sites like Zillow and Apartments.com
  2. Set filters by price, town, and bedrooms
  3. Prepare rental resume with income proof and references
  4. Check for landlords accepting Section 8
  5. Tour properties before committing
