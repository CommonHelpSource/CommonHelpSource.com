Middlesex County Housing Matrix

Emergency Shelter

  1. Call 2-1-1 or DHCD for shelter referrals
  2. Explain your housing crisis
  3. Provide ID or documents if required
  4. Work with a caseworker if placed in shelter
  5. Start planning for longer-term housing

Senior Housing

  1. Search for 55+ or 62+ communities
  2. Ask Councils on Aging for support
  3. Apply via CHAMP or through private landlords
  4. Ask about subsidized options
  5. Get help from a local case manager
